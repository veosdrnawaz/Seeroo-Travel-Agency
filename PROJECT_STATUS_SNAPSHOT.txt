5️⃣ ERROR LOG SUMMARY
- Major errors encountered:
  * Python 3.14.2 lacks precompiled binary wheels for `psycopg2-binary` and older `pydantic-core` versions on Windows, causing Maturn/C++ source compilation to fail without VS build tools. Solved by removing `psycopg2-binary` from local requirements and upgrading to `pydantic>=2.10.0` which has native precompiled wheels.
  * SQLAlchemy 2.0.28 asserts unexpected internal metadata attributes on Python 3.14. Resolved by upgrading to `sqlalchemy>=2.0.32` which is certified for Python 3.14.
  * Pydantic schemas lacked `email-validator` dependency. Resolved by adding `email-validator` to the requirements file.
